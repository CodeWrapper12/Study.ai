# Sliding Window & Stack

## Sliding window
For *contiguous subarray/substring* problems. Two flavors:

- **Fixed size:** maintain a window of k, slide and update.
- **Variable size:** expand `right` greedily; shrink `left` while the window is invalid.

```python
def longest_unique_substring(s):
    seen, left, best = set(), 0, 0
    for right, ch in enumerate(s):
        while ch in seen:              # shrink until valid
            seen.remove(s[left]); left += 1
        seen.add(ch)
        best = max(best, right - left + 1)
    return best
```

Core set: Best Time to Buy/Sell Stock, Longest Substring Without Repeating, Longest Repeating Character Replacement, Minimum Window Substring (hard — attempt after the others).

## Stack
For *nesting, matching, and "most recent" relationships*: Valid Parentheses, Min Stack (store running min alongside), Evaluate RPN.

**Monotonic stack** — the senior-level twist: keep the stack sorted to answer *next greater/smaller element* in O(n). Daily Temperatures and Largest Rectangle in Histogram are the canonical pair. If you can explain *why* each element pushes and pops at most once (amortized O(n)), you sound like the architect you are.
