# Trees & Graphs (BFS / DFS)

**This is the round that decides senior loops.** Trees test recursion clarity; graphs test modeling skill.

## Trees
- **DFS recursion template:** handle `None`, recurse left/right, combine. Max Depth, Invert Tree, Diameter (track best via nonlocal), Balanced Tree.
- **BFS with `deque`:** Level Order Traversal, Right Side View.
- **BST property:** Validate BST (pass min/max bounds down — the classic trap is checking only parent-child).
- **Hard tier:** Lowest Common Ancestor, Serialize/Deserialize.

## Graphs
First skill: *recognize* the graph. Grids, prerequisite lists, and social connections are all graphs.

```python
def bfs(start, graph):
    from collections import deque
    q, seen = deque([start]), {start}
    while q:
        node = q.popleft()
        for nxt in graph[node]:
            if nxt not in seen:
                seen.add(nxt); q.append(nxt)
```

- **Grid flood fill:** Number of Islands, Max Area of Island (DFS + mark visited).
- **Topological sort:** Course Schedule — cycle detection via in-degrees (Kahn's) or DFS colors. Explain it as "dependency resolution," something you've lived with DI containers.
- **Shortest path unweighted:** BFS. Weighted: Dijkstra with `heapq` (know it conceptually; rarely coded in screens).
