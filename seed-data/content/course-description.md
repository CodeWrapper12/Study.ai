## Engineered for one outcome: a senior offer

This is not a tutorial playlist. It is a 15-week operating system for converting a strong senior .NET engineer into a signed Lead/Architect or big-tech offer — built the way a staff engineer would build it: gap-driven, pattern-first, and ruthlessly practical.

**The thesis.** Senior interviews are not knowledge tests; they are *performance under observation*. So every module here trains performance, not trivia: you narrate solutions aloud from week one, you build distributed systems instead of reading about them, and you close the course by running real interview simulations against the same rubrics interviewers use.

### What's inside

- **A daily algorithms track (Module 1)** that runs the entire 15 weeks in parallel — pattern-ordered, spaced-repetition driven, with in-browser Python exercises and a graded simulated screen.
- **A system design method (Module 2)** you can execute under time pressure, calibrated with real capacity numbers and senior-level trade-off language.
- **Hands-on distributed .NET (Modules 3–4):** MassTransit, RabbitMQ, Kafka, sagas and the outbox pattern, gRPC, YARP, polyglot persistence, Kubernetes — the exact stack gap between "senior developer" and "solution architect" in Gulf and big-tech job specs.
- **Cloud architecture & IaC (Module 5)** mapped to AZ-305, with Terraform from zero.
- **AI engineering for .NET (Module 6)** — RAG, agents, embeddings — the differentiator almost no .NET candidate brings.
- **Interview execution (Module 7):** behavioral story banks in STAR form, mock-loop protocols, offer negotiation for UAE/KSA and big-tech compensation structures.

### How it's taught

Every lesson follows a mastery loop: deep article → worked code → "how this breaks in production" → interview lens → exercises → checklist. Quizzes gate your understanding; assignments are graded against rubrics; the code exercises run **in your browser** with instant verification. You finish each module able to *do* the thing, not describe it.

### Honest expectations

This course assumes 5+ years of real engineering experience and ~10 hours/week of focused work. It will not make interviews easy. It will make you the candidate who has rehearsed everything the interview can throw at you — which is the only durable advantage that exists.
